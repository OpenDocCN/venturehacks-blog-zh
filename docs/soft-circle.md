# 爸爸，什么是软圈？

> 原文：<https://web.archive.org/web/venturehacks.com/soft-circle>

[![feld.png](img/09c9227e6e8d577979a80fe1981c9dce.png)](https://web.archive.org/web/20221119090559/http://www.feld.com/blog/)[布拉德菲尔德](https://web.archive.org/web/20221119090559/http://www.feld.com/blog/)最近[提出了一些好的标准](https://web.archive.org/web/20221119090559/http://www.askthevc.com/blog/archives/2008/06/ive-got-angels.php)在你筹集资金的时候区分真正的追随者和挑刺者:

> “虽然你不能通过合同承诺支持投资者(跟随者)，但你通常可以将真正的投资者与反对者(或者更慷慨地说，看涨期权的人)区分开来。承诺支持投资者将允许你在潜在的主要投资者中使用他们的名字，将参与积极的网络活动，并将指定他们愿意投资的具体金额。
> 
> 这些支持投资者通常被称为“软圈”——你从他们那里得到了承诺，但这并不具有法律约束力。**一个软圆圈将始终有一个美元数额与之相关**[强调是后加的]。”

真正的追随者也会对潜在的潜在客户进行有力的介绍。
