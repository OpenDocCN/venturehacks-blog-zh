# 在签署条款清单之前讨论你的计划

> 原文：<https://web.archive.org/web/venturehacks.com/signing-term-sheet>

昨天，我和一位准备签署投资意向书的企业家头脑风暴，讨论了一系列要做的事情。

在搜索了 Venture Hacks 的档案后，我意识到我们的一篇帖子已经涉及到了这个问题:[在签署投资意向书之前，我们应该做多少尽职调查？](/web/20221006043042/https://venturehacks.com/articles/diligence)我认为这篇文章基本上经得起时间的考验——问题解决了。

我修改了帖子，为潜在投资者增加了这个问题:

> "你同意我们未来两/三/四个季度的计划吗？"

在签署条款清单之前讨论这一点有几个好处:

1.  **你会了解与投资者**一起工作是什么感觉——在你与他共度公司生活之前。如果你不喜欢和他一起工作，他可能不是合适的终身丈夫。
2.  **你发现你的投资者是否同意你的计划**。如果他不同意你的计划或者你不同意他的修改，你为什么要他加入公司？你真的要让一个不同意你计划的人进入董事会吗？
3.  **在之前达成方案*融资是[规范杠杆](/web/20221006043042/https://venturehacks.com/articles/bargaining-for-advantage)*** 。如果你的投资者想在未来改变计划，你可以要求他证明改变的合理性:“我们同意了一项计划，自从我们同意了一项计划以来，情况发生了怎样的变化，为什么这要求我们改变计划？”
